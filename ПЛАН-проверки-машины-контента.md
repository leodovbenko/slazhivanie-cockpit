# План ручной проверки — «Машина контента» (концепт)

Файл: `site/proto-content-machine.html` · в проде: **slazhivanie.ru/proto-content-machine.html**
Смотреть с телефона (концепт рисовался под мобилу). Кидай `#экран` в конец ссылки, чтобы попасть сразу на нужный экран: `#ideas #channels #queue #result #how #board #fanout`.

Что сделано в эту сессию (сверх прошлого): всплывающие подсказки «что это» (кружок ⓘ), вычитка копирайта, единый «ты» вместо смеси «ты/вы», два фикса поведения. Ниже — что пройти руками.

---

## 1. Подсказки ⓘ (главное новое)
Тапни каждый кружок — должен всплыть тёмный пузырёк с объяснением простым языком, стрелка смотрит на кружок, текст помещается в экран. Повторный тап или тап мимо — закрывает. Прокрутка — тоже закрывает.

- [ ] **Идеи** — в первом абзаце два ⓘ: «памяти бизнеса» и «вечернего разбора дня».
- [ ] **Раскладка** (тап «Разложить по каналам» у любой идеи) — ⓘ у заголовка «Раскладка по каналам»; и в каждой карточке выхода ⓘ у строки «публикуется сам — доверие заработано» / «жду одобрения».
- [ ] **Каналы** — ⓘ у «Голос автора» в подзаголовке.
- [ ] **Очередь** — ⓘ у «по слотам расписания».
- [ ] **Результат** — ⓘ у «Серия без пропусков».
- [ ] **Как работаю** — ⓘ в шаге 4 у «доверие уже заработано».
- [ ] **Все потоки** — ⓘ у слова «потоков».

Проверь на узком экране (портрет телефона): пузырёк не должен вылезать за край — прижимается к краю, но остаётся видимым целиком.

## 2. Идеи (вход)
- [ ] Четыре свежие идеи с категорией (🏆/💡/🙅/✨) и строкой «↳ откуда».
- [ ] «Написать / Надиктовать / Из разбора дня» — показывают тост «В проде: …».
- [ ] «Не моё» у идеи — идея исчезает, тост «Гермес запомнит…».
- [ ] «Разложить по каналам · N →» — открывает раскладку. N совпадает с числом включённых каналов потока.

## 3. Раскладка (герой)
- [ ] Сверху закреплена сама идея (полоса слева фиолетовая).
- [ ] Ряд чипов каналов: включённые подсвечены. Тап по чипу — выход добавляется/убирается из списка ниже, счётчик внизу пересчитывается.
- [ ] Строка приватности: у идеи с меткой (напр. «зум с клиентом») — жёлтая «было имя клиента, во всех выходах его нет»; у остальных — зелёная.
- [ ] Карточки выходов: текст под каждый канал, медиа-строка, полоса лимита (у Instagram-поста лимит 300 — заполнение должно быть «горячим»/жёлтым, если текст длинный).
- [ ] Карусель Instagram — рисуется слайдами, последний слайд тёмный.
- [ ] «Одобрить и в очередь» — счётчик меняется на «✓ N выходов в очереди», кнопка гаснет, у карточек статус «в очереди · …».
- [ ] Вернись в Идеи — эта идея теперь в «Уже разложены». Открой её снова: кнопка отправки уже неактивна («В очереди ✓»), **дубли в очередь не добавляются** (это чинили).

## 4. Каналы
- [ ] Карточка «Голос и темы потока» вверху: базовый голос, источник, темы, файлы (.md/.docx), кнопки-тосты.
- [ ] Список площадок с тумблерами. Тумблер — включает/выключает канал в потоке (тост), при этом число «N каналов» на Идеях и в раскладке меняется.
- [ ] Тап по карточке (не по тумблеру) — открывает настройку канала.
- [ ] «Добавить площадку» — кнопки Twitter/Threads/… дают тост.

## 5. Настройка канала
- [ ] Заголовок с иконкой и форматом; подзаголовок про «поверх общего голоса».
- [ ] Тумблер «Публикуем сюда», блоки: Формат, Правила площадки (тон под площадку), Примеры ✅/❌ (× удаляет — тост), Файлы, Задания.
- [ ] Ввод в любое поле — тост «В проде: сохраняем …».
- [ ] Открой **LinkedIn** — у него две подвкладки: «Настройка» и «Рост аудитории». Вторая — метрики + «Рост сети» + последние посты. У остальных каналов подвкладок нет.
- [ ] В «Рост сети» текст теперь на «ты» («по твоему портрету… Отправляешь сам») — проверь, что нет «вашему/Отправляете».

## 6. Очередь → Пост
- [ ] Лента строк: иконка канала, заголовок, слот 🕑, статус-чип (ждёт одобрения / одобрено / собирается видео).
- [ ] Тап по строке — открывается пост. «Редактировать» → текст в поле → «Сохранить» — тост, текст обновился в ленте; «Отменить» — не меняет.

## 7. Результат
- [ ] Четыре плитки (идей на входе / выходов / опубликовано / правок).
- [ ] «По каналам за 30 дней» и «Ритм недели» (цель + серия 🔥).

## 8. Как работаю
- [ ] Верхняя полоса из 5 шагов со стрелками (Ловлю момент → … → Пощу).
- [ ] Пять пронумерованных шагов ниже.

## 9. Переключение потоков (проверить внимательно — тут был фикс)
Тап по плашке автора вверху → меню из трёх потоков (Леонид / Уклад ИИ / Aiverix).
- [ ] Выбор другого потока — меняются идеи, каналы, очередь, цифры под него; тост.
- [ ] У «Уклад ИИ» включены другие каналы (tg/li/блог), у «Aiverix» — li/vc. Число каналов в раскладке под каждый поток своё.
- [ ] **Ключевое:** переключи поток, находясь на раскладке / настройке канала / открытом посте — должно **вернуть на верхний экран нового потока** (Идеи/Каналы/Очередь), а не показывать старое содержимое прошлого потока.
- [ ] «Все потоки» — доска с карточками; тап по карточке = переключение на этот поток.

## 10. Мелочи
- [ ] Ссылки-диплинки `#fanout`, `#queue` и т.д. открывают нужный экран сразу.
- [ ] Вкладки внизу шапки подсвечивают активный раздел (раскладка подсвечивает «Идеи», настройка канала — «Каналы», пост — «Очередь»).
- [ ] Вендор публикации (Publora/HeyGen) нигде не всплывает — только «Гермес» и названия площадок.

### На что смотреть особо (слабые места)
- Подсказки на самом узком экране — не режется ли пузырёк.
- Формулировки подсказок — всё ли понятно человеку не из темы (это черновой текст, правь словами).
- «Пощу» в шагах — намеренно разговорно (голос Гермеса). Если режет — скажи, поменяю.
