Стандарт карточки флота

Сейчас карточки собирают четыре разных рендера — у каждого свой словарь статуса и свой набор кнопок. Это прото единого скелета: одни и те же слоты в одном порядке у всех агентов, что бы они ни делали.

  1. Шапка — иконка · имя · тип (часть Первого / отдельный агент) · статус-пилюля.
  2. Статус — два слова, один словарь: ● В работе (нанят/подключён/активен) или Каталог (ещё не в контуре). Больше никаких «подключён / нанят / активна / появится» вразнобой.
  3. Описание — одна–две строки, ровно. Что это за агент.
  4. Плашки-способности — что делает. Оставляем (это самый быстрый скан), но кап 3–4 чипа и единый стиль → карточки выравниваются.
  5. Помощники (суб-агенты) — одна строка, только если есть. Единое место, а не блок то там то тут.
  6. Панель действий — одна главная кнопка (заливка) слева, дальше вторичные (контур) в фиксированном порядке.

Грамматика кнопок (всегда одинаковая):
• В работе: Открыть … (главная дверь агента: чат / @бот / доску) → Настроить → Помощники (если есть).
• Каталог: Нанять (само-найм) или Заказать (мы свяжемся) → Подробнее → (продающий лендинг). Нет лендинга — кнопку не показываем, без битых ссылок.
